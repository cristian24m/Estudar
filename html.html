Explicação das principais partes: Tag Função Define o tipo de documento (HTML5). Contém todo o código da página. Guarda metadados (como título e codificação). Define configurações como charset, autor, palavras-chave, etc. Define o título da aba do navegador. <body> Contém o conteúdo visível da página. quebra da configuração: width=device-width → Diz ao navegador para usar a largura da tela do dispositivo (celular, tablet, etc.), em vez de uma largura fixa. initial-scale=1.0 → Define o nível de zoom inicial da página como 100% (sem zoom). 🧱 Tags Básicas do HTML 📝 <p> – Parágrafo Define um parágrafo de texto. <p>Este é um parágrafo.</p> 🔠 <strong> – Destaque/Ênfase Deixa o texto em negrito e com significado semântico de importância. <p>Esse texto é <strong>importante</strong>.</p> 🔤 <br> – Quebra de linha Usado para pular para a próxima linha sem iniciar um novo parágrafo. <p>Linha 1<br>Linha 2</p> 🔍 <abbr> – Abreviação Exibe o significado da sigla ao passar o mouse. <p><abbr title="HyperText Markup Language">HTML</abbr> é incrível!</p> 📋 Trabalhando com Listas 🔢 <ol> e <li> – Lista Ordenada Cria listas numeradas. <ol> <li>Primeiro item</li> <li>Segundo item</li> </ol> 🔘 <ul> e <li> – Lista Não Ordenada Cria listas com marcadores. <ul> <li>Item A</li> <li>Item B</li> </ul> 🧾 Listas de Definição: <dl>, <dt>, <dd> Usadas para mostrar termos e suas descrições. <dl> <dt>HTML</dt> <dd>Uma linguagem de marcação usada para criar páginas da web.</dd> <dt>CSS</dt> <dd>Uma linguagem de estilo usada para definir a aparência das páginas.</dd> </dl> <dl>: Lista de definição. <dt>: Termo. <dd>: Descrição do termo. 📌 Rodapé: <footer> Usado para adicionar um rodapé na página ou em seções. <footer> <p>© 2025 Cristian Santos.
